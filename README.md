# Kdiss & SAS KOREA 데이터 분석 경진대회 출품작 및 사후 모델 재설계

> 한국데이터정보과학회 & SAS KOREA 제2회 데이터 분석 경진대회 (2인 팀) 출품작과, 대회 종료 후 단독으로 진행한 모델 재설계 실험 결과물 통합 저장

## 개요

6만 명 가상 고객의 3개 이종 테이블을 결합해 **이탈 확률(AUC)** 과 **1년 LTV(RMSE)** 를 동시 예측하는 과제
대회 당시 모델 성능이 AUC 0.794로 수렴하였으나, 사후 실험 진행 결과 **Churn AUC 0.7943 → 0.7982**로 개선

```
baseline/   대회 종료 시점 결과물 (출발점)
improved/   사후 재설계 (진단 → 처방 → 개선)
docs/       상세 실험 보고서 · ERD
```

## 1. 대회 개요 — 문제 정의 & 목표

- **주최:** 한국데이터정보과학회 & SAS KOREA 제2회 데이터 분석 경진대회
- **과제:** 고객별 ① 향후 1개월 내 **이탈 확률**, ② 향후 1년 **LTV 금액** 동시 예측
- **채점:** 두 지표의 가중 평균 — `Score = 0.5 × AUC(Churn) + 0.5 × 1/(1+log(SE(LTV)))`
- **형식:** 2인팀, 주 1회 제출로 5주간 리더보드 경쟁 (총 50팀)

**역할 분배**
- **본인:** 데이터 전처리·EDA, 피처 엔지니어링, 모델링·튜닝, 검증 설계 등 분석 전반 담당
- **팀원:** 초기 데이터 탐색 일부 및 제출물 정리 보조

## 2. 데이터

3개의 이종 테이블과 1개의 정답 테이블을 `customer_id` 기준으로 결합해 사용.

| 테이블 | 내용 | 변수 |
|---|---|---|
| customer_info | 고객 기본정보(데모그래픽) | 7 |
| transaction_history | 리테일 구매이력 (시계열, 2023.07–12) | 6 |
| finance_profile | 금융자산 스냅샷 (신용·예적금·대출·연체 등) | 8 |
| targets | 이탈 여부 · LTV 정답 (학습용) | 2 |

학습 6만 명 / 평가 4만 명(정답 비공개), 평가셋은 Public(2만)·Private(2만)으로 분할.

## 3. 기존 모델 — 대회 종료 시점

대회 5주간 내린 주요 **결정**은 다음과 같음. *(각 결정에 대한 검증은 4·5절에서 다룸.)*

- **피처:** 3개 테이블에서 50개 이상의 파생 피처 생성. 그중 `fin_asset_trend_score`(자산 변동 추세)를 단변량 분리력이 가장 높아 핵심 피처로 판단.
- **모델:** LightGBM · XGBoost 앙상블.
- **타겟:** 이탈(분류)·LTV(회귀)를 각각 모델링, LTV는 2-stage로 설계.
- **검증:** 리더보드 점수를 주된 기준으로 주차별 가설을 반복 검증.

**결과:** 2주차 이후 점수가 **0.794에서 수렴**. 무엇이 점수에 기여하고 기여하지 않는지 분리해 설명할 수 없는 상태로 대회 종료.

## 4. 전환점 — 재설계의 출발점

대회 종료 시점의 미해결 과제는 세 가지로 정리됨.

- 5주간 점수가 정체했으나, **정체의 원인을 규명하지 못함.**
- 핵심 피처·검증 설계 등 주요 결정이 **데이터 검증 없이 직관에 의존**함.
- CV 점수와 리더보드 점수의 괴리를 **추적할 수단이 부재**함.

이에 모델 재설계에 앞서, 기존 의사결정이 올바른 전제 위에 있었는지를 먼저 검증하는 방향으로 사후 실험을 설계함.

## 5. 개선 — 진단 → 처방 → 결과

### 진단: 의심을 데이터로 검증

기존 모델의 결정을 각각 *증상 → 의심 원인 → 확인 방법* 으로 분해해 검증함.

- **피처 과다 → 중요도 왜곡:** null importance(타겟 셔플) z-score 검정 결과, 핵심으로 판단했던 `fin_asset_trend_score`가 실제로는 **noise(z≈-0.9)** 로 확인됨. 단변량 분리력 1위였으나 타 금융 피처와 정보가 중복되어 모델 기여가 0이었던 split-importance 착시 사례.
- **CV-LB 괴리 → 분포 불일치 의심:** adversarial validation으로 검증 → **AUC 0.4995(구분 불가)로 분포 불일치가 존재하지 않음을 반증.** 대회 중 분포 차이의 근거로 삼았던 수치가 집계 오류였음이 드러남.
- **평가 지표 재해석:** 상위팀 실측 Score로 채점식을 역산 → LTV term이 평탄 구간에 위치해, **AUC 0.001 개선이 RMSE 5만 원 개선보다 약 6배 큰 점수 기여**를 가짐을 확인. 역량을 Churn AUC에 집중하는 근거가 됨.

### 처방: 한 번에 하나씩 통제된 실험

- 검증 스킴을 5-fold StratifiedKFold로 고정 — 신뢰 가능한 토대 선확보
- null importance 기반 피처셋(full / denoised / core) **실측 비교** — 추측 아닌 데이터 기반 선택
- 학습 메커니즘이 상이한 모델(CatBoost · RandomForest)을 **앙상블 다양성** 목적으로 도입
- 효과 없던 시도(피처 서브셋 분할, 거래피처 재설계)도 **기각 근거와 함께 기록**

### 결과

| 구분 | Churn AUC |
|---|---|
| 기존 (대회 종료) | 0.7943 |
| **개선 (사후)** | **0.7982 (+0.0039)** |

- 기여도 분해 결과 **하이퍼파라미터 튜닝(+0.0045)** 이 주동력. 단, 피처·검증·신호 정리가 선행된 뒤에야 효과가 발생함.
- 피처셋 비교: full(43개) 0.7963 < **denoised(10개) 0.7975** — 노이즈 피처가 일반화 성능을 저해함을 정량 확인.

상세 진단 과정·단계별 기여도 분해·실패 실험 기록은 보고서 참조.
